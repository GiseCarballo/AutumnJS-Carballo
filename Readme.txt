Autumn Deco es una página de juegos didácticos que funciona con pre-venta; es decir, 
que se adquiere un producto en un carrito ya existente (no es la idea principal crear un carrito en este curso),

se puede ver la idea en https://autumn-deco.netlify.app/

Es por ello que se utilizarán nombres de juegos como productos a lo largo de todo el proyecto como "Tangram", "juegos de ingenio", etc.

La idea final de este proyecto, es crear un calendario, donde se pueda ubicar el código del producto ya adquirido,
y en el calendario se muestre la fecha donde estará finalizado el producto y listo para retirar/enviar.
Cada producto tiene distintos tiempos de producción según dificultad del mismo.
